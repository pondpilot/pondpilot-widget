Basic PondPilot Widget Example

This is the simplest way to use the PondPilot widget. Just add the class pondpilot-snippet to any <pre> tag:

SELECT 
    'Hello from PondPilot!' as message,
    CURRENT_DATE as today,
    CURRENT_TIME as now;

Another Example

You can have multiple widgets on the same page:

-- Generate a sequence of numbers
SELECT * FROM generate_series(1, 10) AS t(num)
WHERE num % 2 = 0;
