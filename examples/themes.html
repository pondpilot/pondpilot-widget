Theming the PondPilot Widget

Register custom themes once and reuse them across snippets. Theme tokens map to CSS custom properties, so colours, typography, and syntax highlighting can be tailored to your brand.

Classic Light (built-in)

The default theme is intentionally understated. No extra configuration is required.

SELECT
  'Classic Light' AS theme,
  'Built-in default look' AS description;

Midnight Neon

Extends the dark palette with high-contrast neon accents – perfect for docs that lean into a terminal aesthetic. 

SELECT
  'Midnight Neon' AS theme,
  'Glow-in-the-dark vibes' AS description;

Sunset Sorbet

Extends the light theme with warm gradients and rounded corners. Great for marketing landing pages or lighter documentation. 

SELECT
  'Sunset Sorbet' AS theme,
  'Soft gradients & friendly contrast' AS description;

Forest Mist

A calm woodland-inspired palette with soft greens and spacious typography.

SELECT
  'Forest Mist' AS theme,
  'Soft greens and fresh air' AS description;

Retro Terminal

A neon-infused dark theme that swaps in IBM Plex Mono for a retro terminal flair.

SELECT
  'Retro Terminal' AS theme,
